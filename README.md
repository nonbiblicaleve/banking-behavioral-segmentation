# Banking Behavioral Segmentation & LTV Engine

Система сегментации банковских клиентов на основе психотипов и расчет их пожизненной ценности (LTV). Проект объединяет поведенческую психологию и финансовую аналитику.

## Особенности проекта
В отличие от стандартных моделей сегментации, данный алгоритм учитывает **психоэмоциональные маркеры** поведения:
- **neuro_score**: Индекс тревожности (частота транзакций и снятия наличных).
- **conscientiousness**: Индекс финансовой дисциплины.
- **risk_appetite**: Склонность к рискованным операциям.

### Уникальный сегмент: "Impulsive Intuitive"
В модель включен авторский психотип **"Импульсивный интуит"** — это сегмент пользователей, преимущественно молодого возраста, которые:
- Игнорируют сложные инвестиционные инструменты.
- Принимают решения на основе интуиции (высокий риск при низкой дисциплине).
- Нуждаются в специфических продуктах (крипто-активы, высокорисковый трейдинг).

## Технологический стек
* **Язык**: Python 3.x
* **Базы данных**: SQL (SQLite) — использование CTE и сложных CASE-условий.
* **Библиотеки**: 
    * `Pandas` & `NumPy` — обработка данных.
    * `Seaborn` & `Matplotlib` — визуализация.

## Визуализация результатов
Проект генерирует отчеты, которые помогают банку увидеть:
1. **Распределение типов**: Кто составляет ядро клиентской базы.
2. **Доходность**: Какой психотип приносит максимальный LTV.



## Структура репозитория
- `analysis.ipynb` — основной ноутбук с кодом и комментариями.

## Как запустить
1. Откройте `analysis.ipynb` в Google Colab.
2. Запустите все ячейки.
3. Система сгенерирует синтетические данные (1000 записей), проведет SQL-анализ и выгрузит отчет.
